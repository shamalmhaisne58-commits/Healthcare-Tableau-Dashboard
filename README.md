# 🇪🇺 European Sales Dashboard (Tableau)

##  Overview
This project analyzes sales performance across European countries.

## Objectives
- Track revenue & profit
- Compare country performance
- Identify top products
- Analyze trends

##  KPIs

- Total Profit
- Profit Ratio
- Total Order

## Insights
- Top-performing countries identified
- High-performing product categories found
